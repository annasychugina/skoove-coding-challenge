# Test Assignment

### How to start

``
yarn 
``

### IOS
```
cd ios
bundle install (optional)
pod install
yarn ios
```

### Android
```
yarn android
```

if some errors with paths in Android, should:

```
yarn start --reset-cache

cd android
./gradlew clean
yarn android
```

### How to test

```
yarn test
```
